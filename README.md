# CentralStatic

Cut through the static. AI-curated radio with VibeConnect integration for authentic music discovery and event-based exclusive drops.

## Vision

CentralStatic is more than a radio platform - it's a complete ecosystem that connects music discovery, real-world events, and economic empowerment through blockchain technology.

### Features

- **AI-Curated Radio**: Mood-based music curation without corporate algorithm manipulation
- **VibeConnect Integration**: Discover local events, connect with attendees, unlock exclusive music drops
- **SoundCloud-First**: Supporting independent artists and unfiltered music
- **Token Economy**: Earn rewards for authentic participation and bringing good vibes
- **Artist Portal**: Direct connection between artists and fans without gatekeepers

## Tech Stack

- **Frontend**: React 18 with Tailwind CSS
- **Backend**: Python FastAPI (planned)
- **Blockchain**: Polygon for NFTs and token rewards
- **Music**: SoundCloud API integration
- **AI**: OpenAI API for personality matching and curation

## Quick Start

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/central-static.git
   cd central-static
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm start
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

### Build for Production

```bash
npm run build
```
